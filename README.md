## Información del Proyecto

- **Lugar de aplicación:** San Salvador, El Salvador (escuelas públicas)
- **Duración:** Marzo 2024 – Mayo 2024
- **Entidad:** Asociación de Matemáticos de El Salvador / Universidad de El Salvador  
- **Tipo de proyecto:** Investigación aplicada en Machine Learning

## Objetivo

Desarrollar un modelo predictivo (Random Forest) para analizar patrones de matrícula escolar utilizando datos geoespaciales.

## Enfoque

El proyecto integra variables espaciales y educativas para mejorar la comprensión del comportamiento de la matrícula en el sistema educativo público.

# predictive-modeling-schools-sv
Modelo Random Forest para predecir matrícula escolar en San Salvador usando datos geoespaciales.
